Goodreads Web Component

Summary

Create a Goodreads Web Component to achieve this use the power and simplification of Polymer 2
This web component will help users to search for books.

Expected results

  1. Organize well your dependencies, use your favorites tools to organize and maintain all code dependencies for vendors
  2. Build a custom element with Polymer 2 to output a small widget for book search in Goodreads. The required custom element is declared in this file. We need you to identify it and to implement it.
  3. To help you with this task a simple PHP connector is attached with this project.
    • Configure a local PHP webserver to process the index.php file
    • To perform a search you just need to do the following request
      http://yourlocalwebserver/index.php?query=example&page=1&field=all
    • The query parameter is the search value
    • The page parameter the pagination value
    • The field parameter is the field where to search, options are title, author and all
    • It should return a json response
    • This small PHP lib has a cache folder as a dependency, make sure you have the correct file permissions
  4. The search request should be done in a self contained custom element
  5. The main custom element should be the orquestrator between any custom elements you decide to create
